# communication_hijacking
Supplementary Materials: Source Accounts and Screenshots (DOI: 10.55682/cdr/skz5-43gg)

SUPPLEMENTARY MATERIALS: SOCIAL MEDIA SOURCES AND SCREENSHOTS

This repository provides the complete supplementary materials associated with the study of Santacaterina et al. (2026), including the full list of source social media accounts and all collected screenshots used in the analysis. These materials accompany and support the findings reported in:
Santacaterina, Donald, Daniel Eerhart, Jason C. Brown, Alex Nelson, and Brian Murphy. “Mapping Communication Hijacking in the Asia-Pacific: Data-Driven Insights into Disinformation Networks.” The Cyber Defense Review 11, no. 1 (2026). https://doi.org/10.55682/cdr/skz5-43gg
